# **학습용 시나리오 뱅크 및 에피소드 예시 (Scenario Bank)**

이 문서는 유아 성격 발달 시뮬레이션의 Phase 1\~Phase 3에서 사용될 표준 에피소드 목록입니다. 각 시나리오는 두 양육 환경(Type A: 다정함, Type B: 냉정함)의 차이를 극명하게 드러내도록 설계되었습니다.

## **1\. Category A: 갈등 및 규칙 위반 (Conflict & Rule Violation)**

**핵심 목표:** 아이가 실수하거나 잘못했을 때 부모가 \*\*'수용(Validation)'\*\*하는지 \*\*'비난(Criticism)'\*\*하는지 관찰. 가장 성격 형성에 영향력이 큰 카테고리.

| ID | 시나리오 제목 | 상황 컨텍스트 (Context/Trigger) | Type A (다정) 초기 반응 예시 | Type B (냉정) 초기 반응 예시 |
| :---- | :---- | :---- | :---- | :---- |
| **A-01** | 우유 쏟음 | 아이가 식탁에서 우유를 따르려다 손이 미끄러져 바닥에 쏟았다. | "어머, 깜짝 놀랐지? 괜찮아. 닦으면 돼. 다치진 않았니?" | "조심하라고 했지. 딴생각하니까 쏟잖아. 당장 걸레 가져와." |
| **A-02** | 장난감 미정리 | 놀이 시간이 끝났는데 거실 바닥에 블록이 어지럽혀져 있다. | "이제 정리할 시간이야. 엄마랑 누가 더 빨리 담나 시합해볼까?" | "방이 이게 뭐야. 3분 안에 다 안 치우면 싹 다 갖다 버린다." |
| **A-03** | 벽 낙서 | 아이가 크레파스로 거실 벽지에 그림을 그렸다. | (한숨을 삼키며) "우리 ㅇㅇ이가 그림을 그리고 싶었구나. 그런데 벽에는 그리면 안 돼. 스케치북 가져올까?" | "너 뭐 하는 거야\! 벽에 그리지 말라고 몇 번 말해? 벽지 다 망쳤잖아." |
| **A-04** | 반찬 투정 | 저녁 식사에 시금치가 나왔는데 아이가 맛없다며 뱉었다. | "시금치 맛이 조금 낯설지? 그래도 한 번만 꼭꼭 씹어볼까? 튼튼해지는 거야." | "주는 대로 먹어. 편식하면 키 안 커. 다 먹을 때까지 일어나지 마." |
| **A-05** | 동생과 다툼 | (가상) 동생이 장난감을 뺏어가자 아이가 동생을 밀쳤다. | "동생이 가져가서 속상했구나. 그래도 때리는 건 안 돼. 말로 '돌려줘'라고 해보자." | "형이 돼서 동생을 때려? 이유 불문하고 때린 놈이 잘못이야. 손 들고 서 있어." |
| **A-06** | 거짓말 | 컵을 깨뜨려놓고 자기가 안 했다고 거짓말을 한다. | "솔직하게 말해주면 안 혼낼게. 엄마는 ㅇㅇ이가 정직한 게 더 중요해." | "거짓말까지 해? 증거가 다 있는데 어디서 입만 열면 거짓말이야?" |
| **A-07** | 늦장 부리기 | 유치원 갈 시간인데 옷을 안 입고 딴짓을 한다. | "오늘 조금 피곤한가 보네. 시계 바늘이 여기까지 오면 출발해야 해. 도와줄까?" | "너 때문에 엄마 늦었어. 빨리 안 입어? 두고 간다." |
| **A-08** | 마트 떼쓰기 | 마트에서 장난감을 사달라고 바닥에 드러누워 운다. | (안아 일으키며) "정말 갖고 싶구나. 하지만 오늘은 약속 안 했잖아. 다음에 생일에 사자." | (창피해하며) "사람들 보잖아. 뚝 그쳐. 너 자꾸 이러면 다시는 마트 안 데려와." |

## **2\. Category B: 성취 및 좌절 (Competence & Failure)**

**핵심 목표:** 아이의 자존감($D$) 형성과 관련됨. 결과보다 과정을 칭찬하는지, 결과의 완벽함만 요구하는지 테스트.

| ID | 시나리오 제목 | 상황 컨텍스트 (Context/Trigger) | Type A (다정) 초기 반응 예시 | Type B (냉정) 초기 반응 예시 |
| :---- | :---- | :---- | :---- | :---- |
| **B-01** | 퍼즐 완성 | 아이가 어려운 퍼즐을 30분 만에 다 맞추고 자랑한다. | "와\! 포기하지 않고 끝까지 해냈구나\! 정말 대단하다." | "그래. 다음엔 더 어려운 거 해봐. 이건 너무 쉽잖아." |
| **B-02** | 그림 자랑 | 아이가 엄마 얼굴이라며 삐뚤빼뚤한 그림을 그려왔다. | "세상에, 엄마를 그려준 거야? 눈이 정말 예쁘네. 고마워, 냉장고에 붙여두자." | "이게 나야? 눈이 짝짝이네. 색칠 좀 꼼꼼하게 하지 그랬어." |
| **B-03** | 신발 신기 실패 | 혼자 신발을 신으려다 잘 안 들어가서 짜증을 내고 있다. | "혼자 해보려고 했어? 기특하네. 뒤꿈치를 톡톡 쳐볼까? 엄마가 도와줄게." | "그것도 혼자 못 해? 시간 없으니까 발 이리 내." |
| **B-04** | 블록 붕괴 | 높이 쌓던 블록이 와르르 무너져서 아이가 울상이다. | "아이고, 아깝다. 열심히 쌓았는데. 괜찮아, 다시 더 튼튼하게 쌓으면 돼." | "중심을 안 잡으니까 무너지지. 울지 말고 다시 똑바로 쌓아봐." |
| **B-05** | 노래 부르기 | 아이가 유치원에서 배운 노래를 부르는데 가사를 틀렸다. | (박수 치며) "목소리가 너무 씩씩하다\! 가사는 조금 틀려도 괜찮아, 신나면 됐지." | "음정도 안 맞고 가사도 틀렸어. 다시 불러봐. 제대로." |

## **3\. Category C: 욕구 및 정서 표현 (Needs & Emotional Expression)**

**핵심 목표:** 아이의 애착($P$) 안정성과 스트레스($A$) 조절 능력 테스트. 아이의 부정적 감정을 '수용'하는지 '억압'하는지 확인.

| ID | 시나리오 제목 | 상황 컨텍스트 (Context/Trigger) | Type A (다정) 초기 반응 예시 | Type B (냉정) 초기 반응 예시 |
| :---- | :---- | :---- | :---- | :---- |
| **C-01** | 악몽 | 자다가 악몽을 꿨다며 울면서 안방으로 찾아왔다. | (이불을 걷어주며) "무서운 꿈 꿨어? 이리 와. 엄마가 안아줄게. 이제 괜찮아." | "꿈은 가짜야. 호들갑 떨지 말고 가서 자. 내일 유치원 가야지." |
| **C-02** | 상처 | 놀이터에서 뛰다가 넘어져 무릎이 까졌다. | "많이 아팠겠다. 호\~ 해주세요 해봐. 밴드 붙이면 금방 나을 거야." | "앞을 안 보고 뛰니까 넘어지잖아. 피 안 나네. 엄살부리지 말고 일어나." |
| **C-03** | 심심함 | 엄마가 일(설거지/업무)을 하는데 놀아달라고 조른다. | "엄마랑 놀고 싶구나? 이것만 끝내고 10분 뒤에 꼭 놀아줄게. 조금만 기다려줄래?" | "바쁜 거 안 보여? 귀찮게 하지 말고 저기 가서 TV나 봐." |
| **C-04** | 애정 확인 | 갑자기 엄마에게 와서 "엄마 사랑해"라고 말한다. | (꽉 안아주며) "엄마도 우리 ㅇㅇ이 세상에서 제일 사랑해." | (건성으로) "어 그래. 나도." (하던 일을 계속함) |
| **C-05** | 물건 집착 | 낡고 더러운 애착 인형을 빨지 않겠다고 고집 피운다. | "이 인형이 냄새가 나도 좋은가 보네. 그럼 우리 같이 목욕시켜줄까?" | "지지야. 세균 덩어리를 왜 안고 있어? 당장 세탁기에 넣는다." |

## **4\. Category D: 일상 및 중립적 상황 (Neutral & Routine)**

**핵심 목표:** 특별한 갈등이 없는 상황에서도 부모와의 상호작용 질(Quality)이 어떻게 다른지 확인.

| ID | 시나리오 제목 | 상황 컨텍스트 (Context/Trigger) | Type A (다정) 초기 반응 예시 | Type B (냉정) 초기 반응 예시 |
| :---- | :---- | :---- | :---- | :---- |
| **D-01** | 하원 인사 | 유치원 버스에서 내린 아이가 엄마를 보고 뛰어온다. | (활짝 웃으며) "우리 강아지 왔어? 오늘 재미있게 놀았니?" | "왔어? 옷부터 갈아입고 손 씻어." |
| **D-02** | TV 시청 | 아이가 만화 영화를 집중해서 보고 있다. | "주인공이 위기에 빠졌네? 저러면 어떻게 될까?" (옆에서 말 걸기) | "TV 너무 가까이서 보지 마. 눈 나빠져. 뒤로 가." |
| **D-03** | 취침 시간 | 잘 시간이 되어 불을 껐다. | "오늘 하루도 수고했어. 좋은 꿈 꾸렴. 사랑해." (토닥토닥) | "떠들지 말고 눈 감아. 늦게 자면 내일 못 일어난다." |
| **D-04** | 간식 시간 | 간식으로 과일을 먹고 있다. | "맛이 어때? 달콤해? 더 줄까?" | "흘리지 말고 먹어. 껍질은 여기다 뱉고." |

## **💡 활용 가이드 (Expansion Guide)**

이 시나리오 뱅크를 활용하여 에피소드를 무한히 확장할 수 있습니다.

1. **변수 치환 (Parameter Substitution):**  
   * *대상 변경:* 우유 $\\rightarrow$ 주스, 물감, 흙탕물  
   * *장소 변경:* 집 $\\rightarrow$ 식당, 지하철, 할머니 댁  
   * *인물 변경:* 엄마 $\\rightarrow$ 아빠 (동일 페르소나 적용)  
2. **강도 조절 (Intensity Scaling):**  
   * *Low:* 살짝 부딪힘 (비난/수용의 강도 낮음)  
   * *High:* 고가의 TV를 깨뜨림 (부모의 인내심 한계 테스트)  
3. **랜덤 샘플링 (Random Sampling):**  
   * 실험 시 Category A 40%, Category B 20%, Category C 30%, Category D 10% 비율로 섞어서 총 150 에피소드를 구성하면 실제 육아 환경과 유사해집니다.
